# Proposta Comercial - Agenda SN Beauty

## Produto

Sistema de agendamento online para estudios de beleza, com fluxo publico, painel administrativo, regras de horario, avaliacoes, backup, exportacao e documentacao profissional.

## Entregaveis

- Site publico responsivo.
- Painel administrativo.
- Regras de conflito e expediente.
- Termos e privacidade.
- Avaliacoes moderadas.
- Exportacao CSV.
- Backup JSON.
- Deploy orientado para Supabase, com Render como ambiente atual e Vercel como alternativa em teste.
- Documentacao completa.

## Implantacao Sugerida

- Configuracao visual e dados do estudio.
- Configuracao dos servicos.
- Deploy.
- Treinamento rapido.
- Homologacao com agendamentos teste.
