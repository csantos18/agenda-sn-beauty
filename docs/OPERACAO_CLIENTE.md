# Operacao Do Cliente - Agenda SN Beauty

## Rotina Diaria

1. Abrir o painel administrativo.
2. Conferir pedidos pendentes.
3. Confirmar ou remarcar horarios.
4. Cancelar quando a cliente desistir.
5. Marcar como concluido apos o atendimento.
6. Moderar avaliacoes.
7. Exportar CSV quando precisar de relatorio.
8. Baixar backup periodicamente.

## Boas Praticas

- Manter a senha administrativa protegida.
- Conferir dados da cliente antes de confirmar.
- Usar o WhatsApp para combinar pagamento e sinal.
- Evitar alterar agendamento sem registrar motivo.
- Revisar avaliacoes antes de manter publicas.
