# Politica de Privacidade e LGPD - Agenda SN Beauty

Este documento descreve como o Agenda SN Beauty trata dados pessoais no fluxo de agendamento. Ele complementa a pagina publica `/termos` e deve ser revisado juridicamente antes de uso contratual definitivo.

## Dados coletados

O sistema pode registrar:

- nome da cliente;
- telefone/WhatsApp;
- servico escolhido;
- profissional;
- data e horario desejados;
- forma de pagamento informada;
- observacoes enviadas pela cliente;
- status do agendamento;
- avaliacoes publicas moderadas;
- eventos operacionais, como inicio de agendamento e acoes administrativas.

## Finalidade do uso

Os dados sao usados para:

- organizar a agenda do estudio;
- evitar conflito de horarios;
- confirmar, remarcar, cancelar ou concluir atendimentos;
- facilitar contato pelo WhatsApp;
- calcular o sinal de reserva de 20%;
- permitir consulta segura do proprio agendamento;
- gerar backup, exportacao e auditoria administrativa.

## Base operacional

O Agenda SN Beauty nao processa pagamento online. O sinal de reserva e informado no sistema e combinado diretamente pelo WhatsApp.

## Acesso aos dados

O painel administrativo exige login protegido. Dados como telefone, historico completo, exportacao, backup e auditoria ficam restritos a usuarios autorizados.

A consulta publica deve exibir apenas as informacoes necessarias para a cliente confirmar o proprio agendamento.

## Retencao

Os dados podem ser mantidos enquanto forem necessarios para historico operacional, atendimento, auditoria, backup ou obrigacao administrativa do estudio.

## Seguranca

O projeto aplica:

- sessao administrativa protegida;
- cookie `HttpOnly`;
- rate limit;
- CSP e headers de seguranca;
- bloqueio de arquivos internos no navegador;
- armazenamento de chaves sensiveis apenas no servidor;
- auditoria de eventos importantes.

## Direitos da titular

A cliente pode solicitar correcao, consulta ou exclusao de dados diretamente pelo canal de atendimento do estudio, respeitando necessidades operacionais, historico e obrigacoes legais.

## Contato

Canal indicado no app: WhatsApp do estudio.
